## 📝 نبذة عن المشروع
**وكيل الأزهريين الذكي** هو تطبيق ويب تعليمي، خيري، ومستقل تماماً، صُمم خصيصاً لخدمة طلاب ومعلمي الأزهر الشريف بالاعتماد على أحدث تقنيات الذكاء الاصطناعي. 

يقوم التطبيق بتحليل وقراءة كتب المعهد ومقررات المادة بصيغة **PDF** عبر نموذج **Gemini 1.5 Flash** صاحب الذاكرة المليونية، ويقدم بوابات تفاعلية مخصصة تدمج الشرح الأكاديمي، وتوليد أسئلة البوكليت، والتحضير التربوي، مع دعم كامل للميزات الصوتية لتسهيل الحفظ والمذاكرة.

## ⚙️ البنية البرمجية والملفات التأسيسية

المشروع مبني بهيكلية خفيفة وسريعة جداً تعتمد على الملفات التالية:
1. `app.py`: المحرك البرمجي الأساسي وتصميم الواجهة الرسومية عبر Streamlit.
2. `requirements.txt`: يحتوي على الحزم والمكتبات الحيوية المستدعاة (`google-generativeai`, `pypdf`, `gTTS`).
3. `.streamlit/secrets.toml`: وعاء التشفير الآمن لحفظ مفتاح الـ API الخاص بـ Google AI Studio محلياً وأونلاين.

## 🚀 طريقة التشغيل المحلي (للـ Developers)

إذا كنت ترغب في تشغيل واجهة الوكيل محلياً على جهازك أو عبر تطبيق **Termux**:

1. قم بعمل استنتاج للمستودع والدخول إليه:
```bash
git clone [https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git](https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git)
cd YOUR_REPO_NAME

```
 2. قم بتثبيت المكتبات المطلوبة:
```bash
pip install google-generativeai pypdf gTTS

```
 3. قم بتشغيل السيرفر المحلي عبر Streamlit:
```bash
streamlit run app.py

```
